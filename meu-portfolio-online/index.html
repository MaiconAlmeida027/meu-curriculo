Sobre Mim

Bem-vindo(a) ao meu espaço! Sou um futuro profissional de Análise e Desenvolvimento de Sistemas, com um entusiasmo particular pela Inteligência Artificial.

Meu objetivo é desenvolver soluções tecnológicas que sejam não apenas inovadoras e eficientes, mas também seguras e inteligentes.

Minhas principais competências incluem Python, onde crio lógica e automação; Banco de Dados (SQL/MySQL), para estruturar e gerenciar informações complexas; e Cybersegurança, garantindo a integridade e proteção dos sistemas.

Minha abordagem é prática e orientada a resultados, buscando sempre a melhor forma de aplicar o conhecimento em projetos relevantes.

Convido você a explorar meu portfólio para ver como aplico essas habilidades e a descobrir minha visão para o impacto da tecnologia no dia a dia. Estou aberto a novas conexões e oportunidades para colaborar em desafios que impulsionem a inovação.

Meu Portfólio

Aqui você encontrará uma amostra dos trabalhos e projetos desenvolvidos ao longo da minha formação em Análise e Desenvolvimento de Sistemas. Cada item reflete minha evolução e a aplicação prática dos conceitos estudados.

Desenvolvimento de Página Web Estática (HTML/CSS)

Descrição: Este projeto envolveu a criação de uma página web estática, utilizando HTML semântico para a estrutura do conteúdo e CSS puro para estilização. O foco foi na organização do layout, seleção de fontes e cores para garantir uma experiência visual agradável e responsiva (com o que você já fez, pode mencionar isso).

Competências Demonstradas: HTML5 (estrutura semântica), CSS3 (seletores, modelo de caixa, flexbox/grid para layout básico, responsividade simples), boas práticas de design web.

Manipulação de Dados com JavaScript (Exercício/Trabalho da Disciplina)

Descrição: Um exercício prático que explorou a manipulação de arrays e objetos em JavaScript para processamento de dados. Desenvolvi funções para adicionar, remover e pesquisar elementos em estruturas de dados, aplicando conceitos de lógica de programação.

Competências Demonstradas: JavaScript (tipos de dados, arrays, objetos, métodos de manipulação), lógica de programação, resolução de problemas.

Introdução à Estrutura de Banco de Dados (SQL/MySQL)

Descrição: Trabalho focado na criação de esquemas de banco de dados, definição de tabelas e relacionamentos, e execução de consultas básicas (CRUD) utilizando SQL. Este projeto solidificou a compreensão sobre modelagem de dados e a importância da integridade das informações.

Competências Demonstradas: Modelagem de dados, SQL (DDL, DML), MySQL (ou outro SGBD utilizado), conceitos de banco de dados relacional.

Estou em constante aprendizado e ansioso(a) para aplicar e expandir essas habilidades em novos desafios e projetos.
