Sobre Mim

Bem-vindo(a) ao meu espaço! Sou um futuro profissional de Análise e Desenvolvimento de Sistemas, com um entusiasmo particular pela Inteligência Artificial.

Meu objetivo é desenvolver soluções tecnológicas que sejam não apenas inovadoras e eficientes, mas também seguras e inteligentes.

Minhas principais competências incluem Python, onde crio lógica e automação; Banco de Dados (SQL/MySQL), para estruturar e gerenciar informações complexas; e Cybersegurança, garantindo a integridade e proteção dos sistemas.

Minha abordagem é prática e orientada a resultados, buscando sempre a melhor forma de aplicar o conhecimento em projetos relevantes.

Convido você a explorar meu portfólio para ver como aplico essas habilidades e a descobrir minha visão para o impacto da tecnologia no dia a dia. Estou aberto a novas conexões e oportunidades para colaborar em desafios que impulsionem a inovação.

Meu Portfólio: Aplicações e Aprendizados

Aqui você encontrará uma amostra dos conhecimentos e habilidades que estou desenvolvendo ativamente em minha formação em Análise e Desenvolvimento de Sistemas. Cada item reflete minha evolução e a aplicação prática dos conceitos estudados em sala de aula.

Desenvolvimento Web Essencial (HTML/CSS)

Descrição: Este projeto em andamento envolveu a construção de uma página web estática, utilizando HTML semântico para estruturar o conteúdo de forma lógica e acessível. A estilização foi realizada com CSS puro, focando na organização do layout, seleção de fontes e cores, e na implementação de princípios de responsividade básica para garantir uma experiência visual agradável em diferentes dispositivos. Este site que você está navegando é um exemplo prático desse aprendizado.

Competências Demonstradas: HTML5 (estrutura semântica, acessibilidade), CSS3 (seletores, modelo de caixa, layout responsivo básico), boas práticas de organização de código web.

Modelagem e Consulta de Dados (MER e SQL)

Descrição: Durante minhas aulas de Banco de Dados, pratiquei a criação de Modelos Entidade-Relacionamento (MER) para projetar a estrutura de bancos de dados de forma eficiente. Além disso, desenvolvi habilidades em SQL (Structured Query Language), realizando operações de busca e manipulação de dados em tabelas. Isso inclui a criação de tabelas de busca para otimização e recuperação de informações específicas.

Competências Demonstradas: Modelagem de Dados (MER), SQL (consultas, inserção, atualização, deleção), compreensão de estruturas de banco de dados relacionais, lógica de recuperação de dados.

Introdução à Lógica de Programação com JavaScript

Descrição: Este trabalho prático explorou os fundamentos da lógica de programação e a manipulação de dados utilizando JavaScript. Desenvolvi pequenos scripts para processar informações, manipular arrays e objetos, e aplicar conceitos de controle de fluxo e funções. O foco foi em construir uma base sólida para futuras aplicações interativas.

Competências Demonstradas: JavaScript (tipos de dados, variáveis, operadores, estruturas de controle, funções), lógica de programação, resolução de problemas algorítmicos.

Estou em constante aprendizado e ansioso para aplicar e expandir essas habilidades em novos desafios e projetos que moldarão o futuro da tecnologia.
